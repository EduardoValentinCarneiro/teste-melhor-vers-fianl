# Raiz Up

Projeto desenvolvido para a subcategoria 3 do Concurso Agrinho 2026, com base no tema:

`Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente`

## Conceito

O site apresenta uma narrativa visual sobre a relação entre campo e cidade, destacando como
tecnologia, manejo responsável, uso inteligente da água e consumo consciente podem fortalecer o agro
sem romper o equilíbrio ambiental.

## Interações implementadas

- Galeria com fotos e imagens da pasta `img`
- Simulador de equilíbrio com leitura dinâmica de cenário
- Seção de dados reais com fontes oficiais
- Linha do tempo interativa sobre evolução do agro sustentável
- Menu lateral de configuração com tema claro/escuro
- Player de música com play, pause e volume
- Sons ambientais em arquivos de áudio
- Acessibilidade com ajuste de fonte, modo leitura e leitura em voz alta
- Mensagens contextuais por horário
- Cursor temático com resposta visual no clique
- Painel interativo da propriedade com hotspots sobre solo, água, energia, biodiversidade e cidade
- Quiz educativo com pontuação final
- Sistema de conquistas salvo no navegador
- Recorte regional sobre o Paraná
- Gerador de manifesto personalizado
- Mural local com persistência no navegador

## Estrutura

- `index.html`: estrutura principal do site
- `style.css`: identidade visual, animações e responsividade
- `script.js`: interações, simulador, painel interativo, quiz, acessibilidade, conquistas e persistência local
- `img/`: favicons, imagens da galeria e imagem do painel interativo
- `musica/`: pasta preparada para a trilha sonora do projeto

## Publicação

O projeto foi preparado para publicação estática em GitHub Pages ou Vercel.

Checklist recomendado para a inscrição:

1. Deixar o repositório público no GitHub.
2. Preencher o campo `About` do repositório com a descrição do projeto.
3. Adicionar o link publicado do GitHub Pages ou Vercel no campo `Website`.
4. Adicionar o topic oficial `agrinho` nos topics do repositório.
5. Conferir se `index.html`, `style.css` e `script.js` estão carregando sem erros.

## Registro de apoio por IA

Para manter transparência com a rubrica do concurso, este projeto considera apoio de IA na criação da base visual,
da estrutura HTML, das interações e dos textos iniciais.

Descricao do prompt utilizado como base:

`Criar um site para o Agrinho 2026, com o tema Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente, identidade visual própria, seções educativas, interações acessíveis e foco em pontuação máxima na rubrica da subcategoria 3.`
